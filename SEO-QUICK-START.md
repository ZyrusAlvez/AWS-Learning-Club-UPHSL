# Quick SEO Action Items

## Immediate Actions (Do These Now!)

### 1. Submit to Google Search Console
**Priority: CRITICAL**

1. Visit: https://search.google.com/search-console
2. Click "Add Property"
3. Enter: `awslearningclub.uphsl.edu.ph`
4. Verify ownership (use HTML file upload or DNS method)
5. Submit sitemap: `https://awslearningclub.uphsl.edu.ph/sitemap.xml`

### 2. Submit to Bing Webmaster Tools
**Priority: HIGH**

1. Visit: https://www.bing.com/webmasters
2. Add your site
3. Submit sitemap

### 3. Request Indexing for Key Pages
**Priority: HIGH**

In Google Search Console, request indexing for:
- https://awslearningclub.uphsl.edu.ph/
- https://awslearningclub.uphsl.edu.ph/membership
- https://awslearningclub.uphsl.edu.ph/verify

### 4. Social Media Optimization
**Priority: MEDIUM**

Update Facebook page:
- Add website link prominently
- Post about the website launch
- Tag all team members
- Use hashtags: #AWSLearningClub #UPHSL #CloudComputing

### 5. Get Backlinks
**Priority: HIGH**

Request links from:
- UPHSL official website
- UPHSL social media pages
- Team members' LinkedIn profiles
- Team members' personal websites/portfolios

### 6. Create Google Business Profile (Optional)
**Priority: MEDIUM**

If you have a physical office/meeting location:
1. Create Google Business Profile
2. Add location, hours, photos
3. Link to website
